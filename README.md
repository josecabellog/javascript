# Fundamentos de JavaScript

Este repositorio contiene ejemplos básicos de los fundamentos de JavaScript.

## Contenido

El proyecto cubre los siguientes temas:

- **Variables**: Declaración y uso de variables (`js/01-variables.js`)
- **Arrays**: Manejo de arreglos (`js/02-arrays.js`)
- **Objetos**: Creación y manipulación de objetos (`js/03-objects.js`)
- **Condicionales**: Estructuras de control de flujo (`js/04-conditionals.js`)
- **Ciclos**: Bucles y repetición (`js/05-for_loops.js`)
- **Funciones**: Definición y uso de funciones (`js/06-functions.js`)
- **Métodos**: Métodos en objetos (`js/07-methods.js`)

## Estructura

- `index.html`: Página principal del proyecto.
- `index.css`: Estilos básicos.
- `js/`: Carpeta con los scripts de ejemplo.

## Uso

Puedes abrir el archivo `index.html` en tu navegador y revisar la consola para ver la salida de los scripts, o examinar el código fuente en la carpeta `js/`.
